How to Test the Extension:

  1. Color Picker: Click the extension icon, then "Pick Color", and click anywhere on this page
  2. Area Snap: Click "Snap Area" and drag to select a rectangular area
  3. Font Inspection: Click "Inspect Page Fonts" to see all fonts used
  4. Context Menu: Right-click anywhere and use the context menu options
  5. Color Generation: Use the color picker to generate complementary palettes
